# JINSTORY

JINSTORY는 php를 처음 공부하면서 제가 즐겨 찾는 사이트를 참고하여 만들어본 여성의류 쇼핑몰입니다.

# 사용 기술
* Language : PHP, html
* OS : CentOS
* Server : Apache
* Tool : PhpStorm

# 주요 기능
1. 오늘 하루 열지 않음 팝업창
2. 회원가입 & 로그인 & 회원정보 수정
3. Q&A (질문 등록, 수정, 삭제)
4. 카테고리별 상품 목록 보기 & 상품 상세페이지 보기
5. 관리자의 상품 관리 기능 (상품 등록, 수정, 삭제)

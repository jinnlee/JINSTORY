# JINSTORY

JINSTORY는 친구가 쇼핑몰을 운영한다는 소식을 듣고 
<br>제가 즐겨 찾는 사이트를 참고하여 만들어본 여성의류 쇼핑몰입니다.

# 주요 기능
1. 오늘 하루 열지 않음 팝업창 (쿠키이용)
2. 회원가입 & 로그인 & 회원정보수정
3. Q&A (질문 등록, 수정, 삭제)
4. 카테고리별 상품목록보기 & 상품상세페이지보기
5. 관리자의 상품관리기능 (상품등록,수정,삭제)
